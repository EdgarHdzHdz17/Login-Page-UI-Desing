## Run Locally

Clone the Login-Page-UI-Desing

```bash
  git clone https://github.com/EdgarHdzHdz17/Login-Page-UI-Desing.git
```

Go to the project directory

```bash
  cd Login-Page-UI-Desing
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
